All programming languages here are at least at version 1.0 and are totally capable to be used in production environments.

## Obscure Programming Languages
- [Raku](https://raku.org/) - Imagine every syntactic sugar a programming language could have. Raku will have it and some more.
- [Unison](https://www.unison-lang.org/unison-1-0/) - A programming language created for distributed programming. With the big idea to hash identifiers and identify them by its hash not by its name.
